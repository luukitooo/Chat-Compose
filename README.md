# Chat-Compose

clean architecture MVI chat application with dynamic colors and simple UI. You can create an account and send messages to other logged in users.

## What's Used?

- Jetpack Compose
- Compose Navigation
- Dynamic Colors
- Dagger-Hilt
- FireStore
- FirebaseAuth
